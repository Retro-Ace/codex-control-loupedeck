# Changelog

All notable changes to this project are documented here.

## 1.0.0

- Added seven complete 4×3 workspaces: Command, Agents, Skills, Create, Operate, Personal, and Quick Text.
- Added the premium full-wheel reasoning plugin and a plugin-free Lite fallback.
- Added 96 custom action icons, editable SVG artwork, eight workspace guides, a master control map, keyboard reference, hero image, and social preview.
- Added exact prompt and control validators, archive checks, a read-only shortcut checker, and privacy auditing.
- Added Full, Lite, plugin, keybinding, checksum, and all-in-one release packages.
- Added complete installation, shortcut, customization, rebuild, troubleshooting, and physical-test documentation.
