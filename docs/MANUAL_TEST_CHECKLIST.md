# Physical Loupedeck CT test checklist

Structural validation cannot prove a physical controller, app focus, firmware, host version, or custom keybinding behaves correctly. Run this checklist after installation.

## Setup

- [ ] Existing Loupedeck profile is backed up.
- [ ] Full or Lite profile is visible and selected for Codex.
- [ ] Full only: the reasoning wheel fills the center display.
- [ ] All ten custom bindings are visible in Codex's Keyboard Shortcuts screen.

## Touchscreen pages

- [ ] Round 1 opens Command; all twelve controls are visible.
- [ ] Round 2 opens Agents; task 1–9, previous, next, and switch work.
- [ ] Round 3 opens Skills; every tile pastes and does not submit.
- [ ] Round 5 opens Create; every tile pastes and does not submit.
- [ ] Round 6 opens Operate; every tile pastes and does not submit.
- [ ] Round 7 opens Personal; every tile pastes and does not submit.
- [ ] Round 8 opens Quick Text; every tile pastes and does not submit.

## Round and square controls

- [ ] Round 4 opens New Chat; Fn+round 4 opens Quick Chat.
- [ ] Fn+round 5 does nothing.
- [ ] Fn+round 6 toggles Fast.
- [ ] Fn+round 7 approves only a controlled test prompt.
- [ ] A/B/C/D open Personal/Agents/Quick Text/Skills.
- [ ] Fn+A increases reasoning and Fn+C decreases reasoning.
- [ ] E declines or cancels only a controlled test prompt.

## Dials and wheel

- [ ] Task History turns left/right and press opens Switch Chat.
- [ ] Transcript Scroll turns in both directions.
- [ ] Interface Zoom turns in both directions and press resets as designed.
- [ ] Compact Reasoning turns down/up and press cycles.
- [ ] System Volume turns in both directions and press toggles mute.
- [ ] Tab Navigation turns in both directions and press performs its assigned action.
- [ ] Center wheel turns left/down, turns right/up, and taps/cycles.
- [ ] Full only: each wheel input shows a short pulse that does not claim to display the current reasoning level.

## Command safety

- [ ] Add Files and Add Photos open their pickers; each picker is canceled.
- [ ] Voice works only while held and stops on release.
- [ ] Dictate starts and stops on separate presses.
- [ ] Plan and Fast visibly toggle.
- [ ] No prompt tile presses Return or submits automatically.

Record the host version, Codex version, controller firmware, Full/Lite choice, failures, and any untested item when reporting results.
